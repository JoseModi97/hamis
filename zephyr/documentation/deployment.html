Zephyr API

<<-deployment->>

if you want to deploy zephyr packages, its just a matter of minute. simply copy your package and upload to the packages folder of zephyr in which you want to run it. no extra configuration is needed.

we are developing a deploy tool for zephyr. with that tool you can simply create "z-ball" - a archived version of a zephyr package, and then simply upload using zephyr control panel.
